Set Suffix

Your name formats are listed on the left. There are a few fields below that descript information about them used for the name format customization.

Is Suffix Academic

Check this if the title is academic. These are doctoral level suffixes so phd, do, md, etc.

Is Suffix FAmilial

Check this if it's jr. sr. etc.

Is Suffix Professional

Check this if the name format is MBA or CPA.

Is Suffix Religious

Check this if the suffix is religious as in a religious order. CSJ, RSM, etc. These are hard to find / figure out, good luck.
